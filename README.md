# 🤖 AI Joke Generator 😂

A modern web application that generates real-time jokes using an API with a beautiful animated AI background.

---

## 🚀 Features
- 🤖 AI-based joke generation (API)
- 🎨 Futuristic animated background
- 💎 Glassmorphism UI design
- 📋 Copy joke to clipboard
- ⚡ Fast and responsive

---

## 🛠️ Technologies Used
- HTML
- CSS (Glassmorphism + Animation)
- JavaScript
- Joke API (https://v2.jokeapi.dev)

---

## 📂 Project Structure
index.html  
style.css  
script.js  

---

## ▶️ How to Run
1. Download or clone this repository  
2. Open the project folder  
3. Run `index.html` in your browser  
4. Click "Generate Joke" 🎉
